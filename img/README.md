# Figure drop-folder — "the internet, as children drew it"

The Beat-2 gallery (`#metaphor` in `site/index.html`) shows framed **placeholder slots**.
Save each figure here under the exact filename below and it appears automatically (the
placeholder hides on image load). **The gallery already prints the provenance under each
plate — keep it.**
